Coding guidelinesÔÉÅ

For code style, we recommend the google style guide.

Pre-commit hooks apply the black code formatting.

For docstrings we use numpy format.

We use ruff for further static code analysis. The pre-commit hooks show errors which you need to fix before submitting a PR.

Last but not the least, we use type hints in our code which is then checked using mypy.
